# 当前版本V0.0.1
## 基于springboot，整合satoken、sqltoy的权限管理系统
由此命名为：sss-rbac-admin

## 启动说明
### 后端
* Main主入口类：sra-bootstrap/src/main/java/com/jwss/bootstrap/SraApplicationBoot.java
### 前端
```bash
$ cd view-vue-ui/sra-vuetify  #切换到前端目录
$ npm install                 #安装依赖  
$ npm run serve               #开发环境  
$ npm run build               #生产环境
```
### 访问地址
- 后端接口文档：http://localhost:7777/sra-api/doc.html  
- 前端登录地址：http://localhost:8080/login
### 备注
1. 必须安装Redis；
2. 目前只支持mysql数据库；

## 项目结构
```
├─logs  ------------------------日志文件  
├─doc  -------------------------文档  
│  └─db  -----------------------数据库脚本  
├─sra-bootstrap  ---------------启动模块  
├─sra-common  ------------------常用模块  
├─sra-config  ------------------配置模块  
├─sra-framework  ---------------框架模块  
├─tools  -----------------------vo和实体类生成工具  
└─view-vue-ui  -----------------前端项目  
```
