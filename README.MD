#### 当前版本V1.0.0

#### 介绍
* 基于springboot，整合satoken、sqltoy的权限管理系统，由此命名为：sss-rbac-admin

#### 软件架构
* 前后端分离，后端采用流行的MVC三层架构

#### 启动说明
1. 运行数据库脚本
2. 启动本地Redis并运行后端服务
3. 安装前端依赖并运行前端项目
##### 数据库脚本
* /doc/db/db_sra_v1-DDL.sql
##### 后端
* Main入口主类：sra-bootstrap/src/main/java/com/jwss/bootstrap/SraApplicationBoot.java
##### 前端
```bash
$ cd view-vue-ui/sra-vuetify        #切换到前端目录
$ npm install   || yarn             #安装依赖  
$ npm run serve || yarn run serve   #开发环境  
$ npm run build || yarn run build   #生产环境
```
##### 访问地址
- 后端接口文档：http://localhost:7777/sra-api/doc.html  
- 前端登录地址：http://localhost:8080/login
###### 备注
1. 必须安装Redis；
2. 目前只支持mysql数据库；

#### 项目结构
```
├─logs  ------------------------日志文件  
├─doc  -------------------------文档  
│  └─db  -----------------------数据库脚本  
├─sra-bootstrap  ---------------启动模块  
├─sra-common  ------------------常用模块  
├─sra-config  ------------------配置模块  
├─sra-framework  ---------------框架模块  
├─sra-system  ---------------系统模块  
├─tools  -----------------------vo和实体类生成工具  
└─view-vue-ui  -----------------前端项目  
```
